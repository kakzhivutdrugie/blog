❓ ВОПРОСЫ И ПРОБЛЕМЫ:

Проблема: Фото не отображаются
Решение: Проверь что папка images/ находится в корне репозитория
          рядом с HTML-файлами

Проблема: Статья не открывается
Решение: Подожди 1-2 минуты после загрузки на GitHub
          (GitHub Pages обновляется не мгновенно)

Проблема: Хочу изменить текст
Решение: Открой HTML-файл в любом текстовом редакторе
          (TextEdit, Notepad, VS Code)
          Найди нужный текст между тегами <p>...</p>
          Измени → сохрани → загрузи на GitHub
